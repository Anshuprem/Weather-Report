# Weather App 🌦️

A simple weather forecast website built using **HTML**, **CSS**, and **JavaScript**.

## 📌 Features
- Current weather conditions: temperature, humidity, wind speed
- City search functionality

## 📁 Files
- `index.html`
- `main.html`
- `style1.css`, `style2.css`
- `script.js`
